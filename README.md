# プログラムの概要
まず、全てのビートルズの歌詞をスクレイピングで取得し、それをScrapyで処理します。さらにそれをword2vecを用いて自然言語処理を施し、単語ベクトルを割り出します。結果として、単語の類似度を測れるようにになっています。

# 画像に出力
またビートルズの全ての歌詞にある単語をリストアップし、可視化できるようにwordcloudを使用しました。
